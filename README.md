# Studio Axis - Site Node.js

## Como Rodar o Projeto

### Pré-requisitos
- Node.js instalado (versão 14 ou superior)
- npm (geralmente vem com Node.js)

### Instalação

1. Descompacte o arquivo `Studio Axis - Site Node.zip`

2. Abra o terminal/prompt de comando na pasta do projeto

3. Instale as dependências:
```bash
npm install
```

### Rodar o Servidor

Para iniciar o servidor, execute:
```bash
npm start
```

O site estará disponível em: http://localhost:3000

### Estrutura do Projeto

- `server.js` - Servidor Express.js
- `public/` - Arquivos estáticos (HTML, CSS, JS, imagens)
- `package.json` - Configuração do projeto e dependências
- `.gitignore` - Arquivos ignorados pelo Git

### Parar o Servidor

No terminal, pressione `Ctrl + C` para parar o servidor.
